## Instalar
### Clonar
En Window$ necesitas [git-scm](https://git-scm.com/) para hacer el siguiente comando:
```
$ git clone http://www.github.com/makmm/lista-de-compras.git
```
### Instalar dependencias
```
$ npm install
```
### Correr
```
$ node index.js
```
